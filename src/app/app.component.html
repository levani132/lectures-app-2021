სემინარის დავალება

  1. დაამატე შესვლის სერვისის გამოძახება.
  2. დაჰენდლე ერორები ისე როგორც ეს რეგისტრაციაში ხდება.
  3. შექმენი user მოდელი ამ ლინკზე მოცემული მოდელის მიხედვით.
  4. შეინახე user auth.service-ში Subject-ის საშუალებით.
  5. გააქრე ღილაკი 'სისტემაში შესვლა' და გამოაჩინე ღილაკი 'გამოსვლა' თუ შენახული გვაქ მომხმარებელი.
